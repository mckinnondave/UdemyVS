Variables and Constants
- int number;
- int Number = 1;
- const float Pi = 3.14f;

Identifiers
- Cannot start with a number (1route), but could be oneRoute
- Cannot include whitespace
- Cannot be a reserved keyword like int (could use @int)
- Use meaningful names - firstName as opposed to fn

Naming Conventions
- For local Vars: Camel Case - int number;
- For constants : Pascal Case - const int MaxZoom = 5;

- To declare a float, you need an f after the number (1.5f);
- To declare a decimal, you add an m at the end (1.5m);

Non Primitive Types:
- String
- Array
- Enum
- Class
